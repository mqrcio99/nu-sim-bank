## 🛠️ Tecnologias Utilizadas

A arquitetura do projeto foi pensada para entregar uma experiência fluida, utilizando ferramentas modernas do ecossistema de desenvolvimento:

*   **⚡ Vite:** Ferramenta de build rápida para um desenvolvimento ágil e otimizado.
*   **📘 TypeScript:** Tipagem estática para um código mais seguro e escalável.
*   **⚛️ React:** Biblioteca componentizada para construção da interface de usuário dinâmica.
*   **🧱 shadcn/ui:** Componentes de interface elegantes, acessíveis e customizáveis.
*   **🎨 Tailwind CSS:** Framework utilitário para estilização rápida e design responsivo.
